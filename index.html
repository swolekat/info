Minigames

Mod Check

How to play: Any non-mod user types out 
modcheck
in chat. Mods have 30 seconds to respond or swolekat EXPLODES.

Word Wizard

How to play: Any sub, VIP, or mod whispers swolekat a message starting with 
!wordwizard
followed by one word (example 
!wordwizard amazing
for the word amazing). A two minute prediction will happen. Chat then has 5 minutes to guess the word that was chosen by just typing it in chat.

Chat Draws Pixels

How to play: in chat type 
!draw x y hexColor
(example 
!draw 0 0 #f00
will make the top left pixel red!). X and Y go from 0 to 7, Hex color can be any valid css color, use #RRGGBB for red green blue.

Recettear Shopping

How to play: Any sub, VIP, or mod whispers swolekat a message starting with 
!shop
followed by a number between 1 and 100000. Chat will have 5 guesses to get within 5% of your number!
